## AdressBookJava Backend

## Command Line 
Compile and run using command line instructions

Build
```
// Windows
gradlew.bat build

// Mac/Linux
./gradlew build
```

Boot Application
```
// Windows
gradlew.bat bootRun

// Mac/Linux
./gradlew bootRun

```
